# GarageKeyPro Design System

Modern design system for car locksmith service website.

## Design Principles

1. **Professional & Trustworthy** - Clean, modern design that builds confidence
2. **Action-Oriented** - Clear CTAs and conversion paths
3. **Mobile-First** - Optimized for on-the-go customers
4. **Visual Hierarchy** - Guide users through content naturally
5. **Speed & Performance** - Fast loading, smooth animations

## Color Palette

### Primary Colors
- **Brand Blue**: `hsl(217, 91%, 60%)` - Trust, professionalism
- **Accent Orange**: `hsl(24, 100%, 50%)` - Urgency, action, CTAs

### Neutral Scale
- Light backgrounds: `hsl(210, 40%, 98%)`
- Dark text: `hsl(222, 47%, 11%)`
- Muted text: `hsl(217, 19%, 38%)`

## Typography

### Font Families
- **Headings**: Plus Jakarta Sans (modern, bold)
- **Body**: Inter (readable, professional)

### Scale
- **Hero**: 40-64px (responsive)
- **H1**: 32-48px
- **H2**: 24-32px
- **H3**: 20px
- **Body**: 16px
- **Small**: 14px

## Components

### Hero Section
- Full-width background image/gradient
- Overlay for text readability
- Large, bold headline
- Clear value proposition
- Dual CTAs (primary + secondary)
- Minimum height: 600px

### Service Cards
- Icon + title + description
- Hover effect: lift + shadow
- Border radius: 16px
- Consistent aspect ratio
- 3-column grid on desktop

### Testimonials
- Customer photo + name + role
- Star rating visual
- Quote text
- Grid or carousel layout
- Authentic, specific quotes

### Lead Form
- Clear field labels
- Large, tappable inputs
- Prominent submit button
- Validation feedback
- Privacy assurance

### Brand Grid
- Logo showcase
- Grayscale on default, color on hover
- Even spacing
- Responsive grid (6 cols → 3 → 2)

## Spacing

### Section Padding
- Mobile: 64px vertical
- Desktop: 96-128px vertical

### Content Gaps
- Between sections: 48-64px
- Between elements: 24-32px
- Between text blocks: 16px

## Effects & Animations

### Hover States
- Cards: lift 4px + increase shadow
- Buttons: darken bg + scale 105%
- Images: zoom 110%

### Transitions
- Duration: 300ms
- Easing: cubic-bezier(0.4, 0, 0.2, 1)

### Loading States
- Skeleton loaders for content
- Pulse animation
- Smooth fade-in when loaded

## Responsive Breakpoints

- **Mobile**: < 640px (1 column)
- **Tablet**: 640-1024px (2 columns)
- **Desktop**: > 1024px (3 columns)
- **Wide**: > 1280px (max-width container)

## Accessibility

- WCAG AA contrast ratios
- Focus indicators on interactive elements
- Semantic HTML structure
- ARIA labels where needed
- Keyboard navigation support
